# Журнал курса валют ЦБ (Currency Journal)

Серверное приложение на базе Spring Boot для автоматического или ручного сбора, хранения и управления данными о курсах валют Центрального Банка Российской Федерации.

## Стек технологий
* Java 25
* Spring Boot
* PostgreSQL
* Liquibase 
* Swagger 
* Jackson

## Основные функции приложения

Приложение реализует следующий набор функций:

### 1. Синхронизация с ЦБ РФ
* **Автоматическая синхронизация** — фоновый запуск по расписанию. Периодичность настраивается (по умолчанию — 1 раз в час).
* **Ручной запуск синхронизации** — инициируется пользователем через REST-запрос. Позволяет актуализировать курсы вне расписания.

### 2. Чтение данных журнала и справочников
* **Чтение журнала курса валют** — получение записей с поддержкой:
  * фильтрации (по валюте, диапазону дат);
  * пагинации (page, size);
  * сортировки (по дате, номиналу, значению курса).
* **Чтение справочника стран-носителей валюты** — получение списка стран.
* **Чтение справочника валют** — получение списка валют с числовым (NumCode) и символьным (CharCode) кодами.

### 3. Редактирование данных
* **Редактирование курса валют** — ручная корректировка значения курса (Value) и номинала (Nominal) в журнале.

## Структура базы данных

Каркас БД и первичный набор справочников формируются с помощью миграций **Liquibase**. Основные таблицы:
* `rate_dict` — справочник валют (содержит NumCode, CharCode, ID с ЦБ).
* `countries` — справочник стран-носителей валют.
* `rates` — журнал курса валют (дата, номинал, значение курса, связь с валютой).
